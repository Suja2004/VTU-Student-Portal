# VTU Student Project & Internship Management Portal

## Overview
A centralized web platform designed to help students track their project and internship-related activities. It provides a simple and user-friendly interface to view student details, project and internship diary logs.

---

## 🎯 Purpose
The portal aims to:
- Simplify tracking of **student academic information**
- Provide clear visibility into **project work** and **Internship**
- Maintain structured **daily diaries** for projects and internships
- Offer a unified dashboard for easy access to all data

---

## 🧩 Key Features

### 👤 Student Information
- Displays essential academic and personal details

---

### 📂 Project Details
- Provides an overview of the student's project

---

### 📂 Internship Details
- Provides an overview of the student's internship

---

### 📘 Project Diary
- Maintains daily logs of project activities
- Supports pagination for navigating multiple entries
- Detailed view available via popup/modal

---

### 💼 Internship Diary
- Tracks daily internship progress
- Similar structure to project diary
- Captures real-world work experience and learnings
